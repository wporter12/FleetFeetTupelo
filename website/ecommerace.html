Shipping An Order!

  1. On the computer, go to shipstation.com
  2. Log In with the already set credientals.
  3. In the "Awaiting Shipment" Tab, select the orders you want to pull
  4. click the print tab and select "Packing Slip"
  5. Collect the packing slips from the computer and locate the shoe in the stock room. It is important that the last 4 numbers on the UPC match the box!
  6. Next, double click the order you have found to print the label
  7. Set your presets (ctrl+z) for single shoe box, (ctrl+l) for big box, and (ctrl+y) for bubble mailer
  8. "Select Create and Print Label" and print from the "Zebra" printer
  9. Tape up the box and attach the label. Set the box in the ecomn area next to the door!
